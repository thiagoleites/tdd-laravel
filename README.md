News Portal (TDD)

Educational project built with Laravel 13 to practice Test-Driven Development (TDD).

The application is developed incrementally using the Red → Green → Refactor cycle, ensuring every feature is driven by automated tests.

Tech stack:
- Laravel 13
- Pest (testing)
- SQLite (in-memory for tests)

Features implemented through TDD:
- User management
- Authentication
- Categories
- News / blog posts
- Public portal
- Admin dashboard
- Authorization and roles
